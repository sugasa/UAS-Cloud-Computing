# Kontribusi Anggota Kelompok

## Anggota 1

### Minggu 1 - Core & Network

* Membuat repository GitHub dan struktur project.
* Mengonfigurasi Docker Compose.
* Menyiapkan WordPress Container dan MariaDB Container.
* Mengimplementasikan Docker Volume untuk persistent storage.

### Minggu 2 - Cache & Object Storage

* Menambahkan Redis sebagai Caching Layer.
* Mengintegrasikan Redis dengan WordPress.
* Melakukan pengujian cache dan monitoring Redis.

---

## Anggota 2

### Minggu 1 - Core & Network

* Mengonfigurasi custom network (frontend-net dan backend-net).
* Melakukan pengujian konektivitas antar-container.
* Menyiapkan dokumentasi dan screenshot deployment.

### Minggu 2 - Cache & Object Storage

* Menambahkan MinIO sebagai Object Storage Layer.
* Membuat bucket `wordpress-media`.
* Mengintegrasikan WordPress dengan MinIO.
* Melakukan pengujian upload media ke Object Storage.
* Memperbarui dokumentasi dan melakukan commit ke GitHub.

---

# Ringkasan Hasil

## Minggu 1

* WordPress berhasil dideploy menggunakan Docker Container.
* MariaDB digunakan sebagai Database Layer.
* Docker Volume digunakan untuk persistent storage.
* Isolasi jaringan diterapkan menggunakan frontend-net dan backend-net.

## Minggu 2

* Redis berhasil diimplementasikan sebagai Caching Layer.
* MinIO berhasil diimplementasikan sebagai Object Storage Layer.
* Media WordPress tersimpan pada bucket Object Storage.
* Monitoring dan dokumentasi berhasil diperbarui pada repository GitHub.
